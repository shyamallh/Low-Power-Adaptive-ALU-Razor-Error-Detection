
# Low-Power-Adaptive-ALU-Razor-Error-Detection
Design of a Low Power Approximate Adaptive ALU with Razor-based Error Detection and Recovery using SystemVerilog.

## About the Project
This project focuses on reducing power consumption in Arithmetic Logic Units (ALU) using approximate computing and Razor-based timing error detection techniques. The design dynamically switches between accurate and approximate modes to improve power efficiency while maintaining reliable operation.

## Technologies Used
- SystemVerilog
- Digital VLSI Design
- RTL Design
- Approximate Computing
- Razor Flip-Flops
- Timing Error Detection
- ModelSim / Vivado

## Features
- Low power ALU architecture
- Adaptive approximate computing
- Razor-based timing error detection
- Error recovery mechanism
- Modular RTL design
- Improved power efficiency

## Applications
- Low power processors
- Embedded systems
- Edge AI hardware
- Energy-efficient computing
